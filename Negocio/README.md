# NegocioGlass v2

Proyecto Android Studio para registrar gastos y ventas por negocio/emprendimiento.

## Incluye
- Kotlin + Jetpack Compose
- Room para persistencia local
- Diseño glassmorphism tipo iPhone
- Negocios separados como Candelas, Chocolates y Postres
- Registro de gastos y ventas
- Balance automático
- Filtros por hoy, semana y mes
- Eliminación de negocios y movimientos

## Cómo abrir
1. Descomprime el .zip.
2. Abre la carpeta `negocioglass_v2` en Android Studio.
3. Espera a que haga Sync con Gradle.
4. Ejecuta en emulador o celular.

## Nota
El archivo `gradle-wrapper.jar` no fue incluido en este entorno. Android Studio normalmente puede regenerar/sincronizar el proyecto al abrirlo.
