
# 📊 NegocioGlass - App de Finanzas

Aplicación móvil desarrollada en Android Studio que permite llevar el control de inversiones y ganancias de manera sencilla y organizada.

## 🚀 Funcionalidades

* Registro de inversiones realizadas
* Registro de ganancias obtenidas
* Visualización de balance general
* Control básico de finanzas personales

## 🛠️ Tecnologías utilizadas

* Java / Kotlin
* Android Studio
* Gradle

## 📂 Estructura del proyecto

El proyecto está organizado en módulos estándar de Android:

* `app/` → Código principal de la aplicación
* `gradle/` → Configuración de dependencias
* Archivos de configuración (`build.gradle`, `settings.gradle`)

## ⚙️ Instalación y uso

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/JordyRetana/Negocio_Finanzas
   ```

2. Abrir el proyecto en Android Studio

3. Sincronizar Gradle

4. Ejecutar en un emulador o dispositivo físico

## 📌 Notas

* No se incluyen archivos de configuración local como `.idea` o `local.properties`
* Asegúrate de tener configurado Android Studio correctamente
